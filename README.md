### 1. Instalación
```
npm i -g @nestjs/cli
```
> Enlace directo de [npm @nestjs/cli](https://www.npmjs.com/package/@nestjs/cli) 

### 2. Comprobar versión
```
nest --versión
```
Resulado: ```10.3.2```

### 3. Instalación del proyecto
Dirigirse al directorio del proyecto:
> Ejemplo: C:/usuarios/.../notas-recordatorio/server 
```
npm install
```

### 4. Iniciar el proyecto
```
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
